# 📚 Sistem Manajemen Perpustakaan Modern

Proyek UAS Praktikum Algoritma dan Struktur Data kelompok 4 menggunakan bahasa C++ dengan metode *Array of Struct*.

## 📂 Struktur Folder
* `src/` : Berisi file source code utama (`main.cpp`).
* `docs/` : Berisi dokumen laporan tugas akhir kelompok.

## 👥 Pembagian Tugas Fungsi (`src/main.cpp`)
* **Anggota 1 (Ketua):** Setup Git, Review Kode, & Integrasi Akhir.
* **Anggota 2:** Fungsi `tambahBuku()`, `hapusBuku()`, & Alur Menu `main()`.
* **Anggota 3:** Fungsi `loadDatabaseBuku()`, `tampilBuku()`, `cariBuku()`.
* **Anggota 4:** Fungsi `pinjamBuku()`, `kembalikanBuku()`.
* **Anggota 5:** Fungsi `sortingBuku()`, `rekomendasiBuku()`, `riwayatBuku()` + Laporan.
